# Teams Room Dashboard

This project is a web application designed to monitor and analyze the performance of Teams Room devices. It provides real-time insights and visualizations to help manage device quality and usage effectively.

## Project Structure

```
teams-room-dashboard
├── index.html          # Main HTML page for the application
├── assets
│   ├── css
│   │   └── styles.css  # Styles for the application
│   └── js
│       └── app.js      # JavaScript code for the application
├── .gitignore          # Specifies files to be ignored by Git
└── README.md           # Documentation for the project
```

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd teams-room-dashboard
   ```

2. **Open index.html**
   Open the `index.html` file in your web browser to view the application.

3. **Styles and Scripts**
   The application uses an external CSS file located in `assets/css/styles.css` for styling and a JavaScript file located in `assets/js/app.js` for functionality.

## Usage

- The dashboard provides various views to analyze device performance, including:
  - Overview
  - Geographic Analysis
  - Trends
  - Comparative Analysis
  - Outlier Detection
  - Network Correlation

## Deployment

This project is configured to be hosted on GitHub Pages. Once the repository is pushed to GitHub, you can enable GitHub Pages in the repository settings to serve the `index.html` file publicly.
